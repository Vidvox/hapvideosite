Welcome to the Official Hap Video Codecs Homepage

The Hardware Accelerated Playback solution for HD and ultra high resolution video.

Hap is a set of video codecs for macOS, Windows and Linux that performs decompression using a computer's graphics hardware, substantially reducing the CPU usage necessary to play video — this is useful in situations where CPU power is a limiting factor, such as when working with multiple high resolution videos in real-time.

Since the initial release in 2013 the open source Hap codecs family has become the standard for high performance high resolution movie playback in the worlds most popular media servers, live video performance and mapping softwares that are powering the visuals running at the worlds largest events.

An Open Standard For Creative Work

Artists need to use their movie files in lots of different tools during the creative process. Nobody wants to be locked into a single platform and having to transcode to lots of different proprietary formats is a pain.

Widely Supported Pro Video Format

The Hap codecs playback natively in over 20 different popular applications and creative coding environments on macOS, Windows and Linux.

Works with Existing Encoding Softwares and Workflows

You can easily use most QuickTime and FFMPEG based exporting softwares to encode movie files to Hap. A free macOS batch exporter is available from VIDVOX and in some cases media servers can automatically transcode loaded media files into Hap.

Alpha Channel Support

For real-time compositing and overlays the Hap Alpha and Hap Q Alpha formats can be used to include the opacity level of each pixel.
