Welcome to the Official Hap Video Codecs Homepage

The Hardware Accelerated Playback solution for HD and ultra high resolution video.

Hap is a set of video codecs for macOS, Windows and Linux that performs decompression using a computer's graphics hardware, substantially reducing the CPU usage necessary to play video â€” this is useful in situations where CPU power is a limiting factor, such as when working with multiple high resolution videos in real-time.

Since the initial release in 2013 the open source Hap codecs family has become the standard for high performance high resolution movie playback in the worlds most popular media servers, live video performance and mapping softwares, powering the visuals running at the worlds largest events.

An Open Standard For Creative Work

Artists need to use their movie files in lots of different tools during the creative process. Nobody wants to be locked into a single platform and having to transcode to lots of different proprietary formats is a pain.

Widely Supported Pro Video Format

The Hap codecs work natively in over 20 different popular applications and creative coding environments on macOS, Windows and Linux.

Works with Existing Encoding Softwares and Workflows

Using the free QuickTime extension you can export to Hap directly from many existing video rendering software and batch exporting applications. This extension is often bundled in the installer for many software packages or downloaded from GitHub.
On the macOS there is a free opensource batch exporter tool from VIDVOX that will encode to and from AVFoundation supported codecs.
FFMPEG can be used from the commandline to encode to Hap.

Optimized For High Performance At High Resolutions

Hap Q and Hap Q Alpha allow for encoding with system specific settings to optimize encodings for ultra high resolution video.

Information For Software Developers

Visit the official Hap GitHub Repository for the specification description, frameworks and sample code to get started with. If you are already using QuickTime, AVFoundation, FFMPEG, LibAV, OpenFrameworks, Cinder or DirectShow for movie playback or encoding it is relatively easy to add native support for the Hap codecs to your own applications.

Open Source

The Hap video codecs are open-source, licensed under a Free BSD License, meaning you can use it in your commercial or noncommercial applications free of charge. This project was originally written by Tom Butterworth who continues to maintain the bulk of the development and was commissioned by VIDVOX in 2012. Additional code and financial support has been provided by other developers and organizations since its initial public release in 2013.
