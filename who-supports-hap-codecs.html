Who Supports The Hap Codecs?

Interoperability Between Platforms

One of the key features of the Hap codecs is the ability to export to a single highly optimized file format that can be used with the most popular media servers, video performance and mapping software.

The following applications and platforms support accelerated playback of the Hap video codecs:

  • Blendy VJ
  • CoGe
  • d3 Media Servers
  • Dataton Watchout
  • Fugio
  • GrandVJ
  • GRoK
  • Isadora
  • MadMapper
  • Millumin
  • Mitti
  • MixEmergency
  • modul8
  • Modulo Player
  • Painting With Light
  • Smode
  • TouchDesigner
  • VDMX
  • Ventuz
  • VPT 7
  • Cinder via Cinder-Hap
  • Max via Hap Video Engine or jit.gl.hap
  • OpenFrameworks via ofxHapPlayer
  • Quartz Composer via CoGeHapPlayer
  • Unity via AV PRO QUICKTIME
  • Vuo

The following applications can play (and/or encode) Hap movies with varying or no GPU-accelaration:

  • AVF Batch Converter
  • FFMPEG
  • MPlayer
  • SMPlayer
